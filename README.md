# preact-mdc-demo

## Overview

This is an example on how to use the material design components from [preact-mdc](https://github.com/BerndWessels/preact-mdc).

## Getting started

### Preparations

- `npm install`

### Development

`npm run dev`

### Production

`npm run build`
